# Gestor de ventas terramoda backend

Backend del gestor de ventas terramoda, este se encarga de hacer las peticiones SQL y de ofrecer los servicios necesarios para obtener la data necesaria.

## Tecnologias

Estas son las tecnologias usadas por el proyecto:
- rustc: 1.86
- mysql: 5.7

## Desarrollo

Primero debes configurar tus variables de entorno puedes saber cuales son leyendo el:
- .env.example

Puedes ejecutar estos comandos para probar el proyecto:

- ejecutar local: docker compose -f docker-compose.local.yaml up -d --build
